# 📝 To-Do App

A simple and interactive To-Do Application built using HTML, CSS, and JavaScript. This project helps users organize and manage their daily tasks efficiently with features such as task creation, completion tracking, deletion, local storage support, and a personalized welcome screen.

## 🚀 Features

* Welcome screen with username
* Personalized greeting based on time of day
* Add new tasks
* Mark tasks as Complete / Undo
* Delete individual tasks
* Clear all tasks
* Save data using Local Storage
* Mobile responsive design
* User-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## 📚 Concepts Practiced

* Variables and Data Types
* Functions
* DOM Manipulation
* Event Handling
* Local Storage
* Conditional Statements
* Arrays and Objects
* Responsive Design

## 🎯 How It Works

1. Enter your name on the welcome screen.
2. Click the Start button to access the application.
3. Add tasks using the input field.
4. Mark tasks as completed or undo them.
5. Delete individual tasks when no longer needed.
6. Clear all tasks using the Clear All button.
7. All tasks and user information are saved automatically using Local Storage.
